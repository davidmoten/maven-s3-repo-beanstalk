maven-s3-repo

ArtifactId: ${project.artifactId}
Version: ${project.version}
  • follows Maven 2+ repository structure (from the base path /repo)
  • supports submission of maven artifacts via HTTP PUT or HTTP POST
  • authenticated by Basic Authentication
  • tested with maven-dependency-plugin (download) and with maven-deploy-plugin (upload)

Authentication

/configuration.properties on the classpath contains the hashed user keys. Run AuthenticationFilter.main in your IDE to generate your key hash and add to hmacs property in /configuration.properties.
